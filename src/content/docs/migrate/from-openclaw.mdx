## 概念映射表

| OpenClaw | → | Hermes Agent |
|---|---|---|
| Skills | → | Skills System |
| Integrations | → | Toolsets + MCP |
| Heartbeats | → | Scheduled Tasks |
| Persistent Memory | → | Memory System |
| Bot Framework | → | Messaging Platforms |

## 迁移步骤（撰写中）

1. 备份 OpenClaw 的 skills 目录和记忆快照
2. 安装 Hermes（参见 [Day 01](/7-days/day-01/)）
3. 一一映射 skill 到 Hermes 的 skill hub 格式
4. 恢复记忆与会话
5. 切换消息平台接入
