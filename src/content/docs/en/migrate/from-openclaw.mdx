## Concept crosswalk

| OpenClaw | → | Hermes Agent |
|---|---|---|
| Skills | → | Skills System |
| Integrations | → | Toolsets + MCP |
| Heartbeats | → | Scheduled Tasks |
| Persistent Memory | → | Memory System |
| Bot Framework | → | Messaging Platforms |

## Migration steps (WIP)

1. Backup OpenClaw skills dir and memory snapshot
2. Install Hermes (see [Day 01](/en/7-days/day-01/))
3. Map each skill to Hermes skill-hub format
4. Restore memory and sessions
5. Switch messaging platform bindings
